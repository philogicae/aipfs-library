# Agent (Backend)

## Requirements

### Tools

- `docker`
- `python v3.12.x` ([pyenv](https://github.com/pyenv/pyenv#installation) recommended)

### Configuration

- Copy `.env.example` to `.env` file and update it

## Installation

```bash
cd agent
./1_install-env.sh
```

## Scripts

```bash	
./scripts/2_test-agent.sh
./scripts/3_deploy-agent.sh
./scripts/4_logs-agent.sh
```
